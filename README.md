# Build Software with AI — Cursor, GitHub & Vercel

A one-page teaching site built with Next.js. It walks through the **Code → Save → Ship** workflow using Cursor, GitHub, and Vercel.

## Local development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build

```bash
npm run build
npm start
```

## Deploy to Vercel

1. Push this project to a GitHub repository.
2. Sign in at [vercel.com](https://vercel.com) and import the repo.
3. Use the default Next.js settings and deploy.
4. Future pushes to `main` trigger new deployments automatically.
